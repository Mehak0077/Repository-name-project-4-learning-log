# Learning Log

A personal learning log application built using HTML, CSS, and JavaScript.

## Features

* Record daily learning notes
* Save key takeaways from each day
* Organize entries by date
* Edit existing entries
* View previous learning records
* Automatically save data using localStorage
* Persistent storage across browser sessions

## Technologies Used

* HTML
* CSS
* JavaScript
* localStorage

## How It Works

1. Select a date.
2. Write everything you learned during the day.
3. Add a key takeaway.
4. Save the entry.
5. Click any previous date to view or edit past learning records.

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* JavaScript Objects
* Local Storage
* Date Handling
* Dynamic Rendering
* Data Persistence
